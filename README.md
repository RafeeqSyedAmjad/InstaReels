# Enreel
📹📹 Enreel

I am introducing Enreel a Video/Reels sharing Web Application.

In this app, user can make their profile, upload a reel/video and can view other users’ posts.

## Features

- Signup
- Login 
- Logout
-  Uploading a reel/video
-  liking
- Commenting
- the following feed
- Profile page


##👨‍💻 Built with

1. ReactJs
2. Firebase
3. Material UI
4. React-DOM
